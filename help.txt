.
NAME
        discord_rename_user - A Discord bot that be used to change a users server nickname

SYNOPSIS
        !r [rename $current_nickname to $new_nickname]
        !r help

EXAMPLES
        !r rename $current_nickname to $new_nickname
            Rename the user with $current_nickname to $new_nickname. If you want to use 'to' in someone's nickname, type a backslash before it like so, '\to'
        !r help
            Outputs this help message
